# المبرمج الأسطوري - ملف تعريف تقني

## نظرة عامة

موقع تعريف لمبرمج أسطوري يعرض السيرة، المهارات، المشاريع، والشهادات، مع دعم لغوي ثنائي (العربية و الإنجليزية) وتوجيه RTL/LTR ديناميكي.

## بنية المشروع

- index.html: صفحة رئيسية كاملة HTML5 مع ترجمة ديناميكية
- assets/css/style.css: أنماط CSS محدثة مع قيم متغيرة للثيم
- assets/js/main.js: منطق التبديل بين اللغات، القائمة المتنقلة، والتنقل السلس
- assets/js/lang.js: ملف مساعد للنُسخ اللغوية (قابل للإزالة أو التوسيع)
- assets/js/project.js: (اختياري لاحقًا) مكوّنات برمجية إضافية
- assets/images/*.svg: بدائل صور placeholder لتعويض الصور الحقيقية
- README-images.md: تعليمات استبدال الصور التمهيدية بالصور الفعلية

## كيفية إضافة الصور

1. ضع ملفات SVG في مجلد assets/images (مثلاً logo-placeholder.svg, hero-placeholder.svg, project1-placeholder.svg).
2. قم بتحديث روابط الصور في index.html إذا رغبت.

## كيفية تغيير النص

- النصوص المعروضة حالياً هي نسخ AR/EN مخزنة في dictionary داخل ملف assets/js/main.js.
- لتغيير النص، عدّل القيم داخل dictionary في قسم AR أو EN.
- استخدم أداة تبديل اللغة لتحديث الواجهة تلقائياً.

## كيف يعمل تبديل اللغة

- يتم تخزين تفضيل اللغة في localStorage (AR/EN).
- عند الضغط على زر تبديل اللغة، يتم تحديث النصوص عبر dictionary بالتزامن مع ضبط اتجاه الصفحة (RTL/LTR).

## كيف تخصّص التصميم

- يمكنك تعديل CSS في assets/css/style.css، وتغيير القيم في :root لثيم مخصص.
- لتغيير النظام الشبكي، عدّل grid-template-columns في الأقسام المناسبة.

## كيف تنشر الموقع

- Netlify: اربط المستودع لديك، اختر فرع النشر، وسيتم إعداد البناء تلقائياً.
- Vercel: اربط المستودع، اضبط إعدادات البناء (npm run build إذا أردت إضافة عملية بناء).
- GitHub Pages: استخدم فرع gh-pages أو إعدادات Pages لاختيار المسار الصحيح.

## معلومات إضافية

- هذا مشروع ثابت (Static Website) مع تصميم حديث ومهارات أساسية في التفاعل.
- الملفات تم تنظيمها بشكل بسيط وواضح لتسهيل التعديل والصيانة.
